(WIP) Site for ["Philately in Troubled Times" project](https://carleton.ca/ccph/2025/philately-in-troubled-times-a-citizen-science-project/).

## Developing

Install dependencies with `npm install`.

Then:
```sh
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

## Building

For production version of app:

```sh
npm run build
```

Then preview with `npm run preview`.
